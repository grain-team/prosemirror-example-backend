# Prosemirror Example Backend

Backend server for Prosemirror Collab.

## Setup

1) Install node dependencies

```bash
$ pushd priv/js && yarn install && popd
```

2) Install Elixir dependencies

```bash
$ mix deps.get
```

## Run

```bash
$ mix run --no-halt
```

With shell:

```
$ iex -S mix
```
